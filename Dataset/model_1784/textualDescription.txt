The domain model 'Book' describes the core structural elements and their relationships within the system.

The model defines 2 main class(es).

'Book_Book' represents an important domain concept.
It has the following attributes: 'authorName', which is of type str; 'title', which is of type str.

'Book_Chapter' represents an important domain concept.
It has the following attributes: 'nbPages', which is of type int; 'title', which is of type str.

Relationships between classes are modeled using associations.
An association connects 'Book_Chapter' and 'Book_Book'. From 'Book_Chapter', the role 'Book_Chapter' has cardinality 0..*, and from 'Book_Book', the role 'Book_Book' has cardinality 0..*.
